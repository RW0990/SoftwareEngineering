#BandSite

#Description
EJS, CSS and Node.js WebApp for a band where events,tickets and merchandise can be bought.

#Technologies used
Node.js
EJS
MongoDB
CSS
Express

#How to install
Clone Repo:
git clone https://github.com/RW0990/SoftwareEngineering

Install:
npm install

Run App in bash/terminal/command line:
nodemon app.js

Open using Express:
http://localhost:3000
